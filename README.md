## Web Programlama Dersi Projesi – Sahaf Web Uygulaması

Bu proje, Web Programlama dersi kapsamında küçük ölçekli bir sahaf için geliştirilen bir web uygulamasıdır.  
Projenin amacı, sahafın kitaplarını web ortamında yönetebilmesini sağlayan temel bir CRUD sistemi oluşturmaktır.

---

## Proje Konusu

Küçük ölçekli bir sahaf için kitap yönetim sistemi geliştirilmiştir.  
Sistem üzerinden sahaf, kitap bilgilerini ekleyebilir, güncelleyebilir, silebilir ve belirli kriterlere göre sıralama yapabilir.

---

## Proje Özellikleri

- Kitap bilgileri yönetimi
  - Kitap adı
  - Yazar adı
  - Fiyat bilgisi
  - Kategori bilgisi
  - Kısa özet açıklaması
- Kitap ekleme işlemi
- Kitap güncelleme işlemi
- Kitap silme işlemi
  - Silme işlemi öncesinde onay mekanizması bulunmaktadır
- Kitapları sıralama
  - Ada göre alfabetik sıralama
  - Fiyata göre küçükten büyüğe sıralama

---

## Kullanılan Teknolojiler

- ASP.NET MVC
- C#
- HTML
- CSS
- JavaScript

---

## Uygulama Yapısı

- MVC mimarisi kullanılmıştır
- Controller katmanı üzerinden kitap işlemleri yönetilmektedir
- View katmanında kullanıcı arayüzü oluşturulmuştur
- Model katmanında kitap verileri tanımlanmıştır

---

## Çalışma Mantığı

- Kullanıcı kitap ekleme formu üzerinden yeni kitap ekleyebilir
- Mevcut kitap bilgileri güncellenebilir
- Kitap silme işlemi sırasında kullanıcıdan onay alınır
- Kitap listesi istenilen kritere göre sıralanabilir

---

## Çıktılar

- Kitapların listelendiği ana sayfa
- Kitap ekleme ve güncelleme formları
- Silme işlemi için onay ekranı
- Sıralama yapılmış kitap listeleri

---

- Web programlama temel kavramlarının pekiştirilmesi
- MVC mimarisinin anlaşılması
- CRUD işlemlerinin uygulanması
amaçlanmıştır.
